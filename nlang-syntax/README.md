# NLANG-SYNTAX README

Provides syntax highlighting for the Nlang programming language in Visual Studio Code.

Nlang is a modern systems programming language with Python-like syntax, static typing, and multiple compilation backends (LLVM IR and C transpilation).

## Features

This extension provides comprehensive syntax highlighting for all Nlang language features, including:

*   Variables and data types
*   Functions
*   Control flow (if, else, while, for)
*   Imports and modules
*   Built-in functions
*   Comments

## Requirements

There are no special requirements for this extension.
